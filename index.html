UI ギャラリー

プロトタイプギャラリー

様々なUIレイアウトとデータダッシュボード案を横断的にまとめています。
画面構成の検討からインタラクション確認まで、カテゴリ別に比較できます。

UI パターン

カタログダッシュボード

基本的なダッシュボード／業務画面の骨組みを扱うコレクションです。ベースレイアウトから詳細タブ構成まで、多様な7案を連続で確認できます。

ギャラリーを開く

UI パターン

ATSダッシュボード

カスタマイズ性の高いレイアウトを集めた応用編です。フィルターの配置やカードグリッドのバリエーションなど、より実装を意識した提案が揃っています。

ギャラリーを開く

データビュー

実績ダッシュボード

実績管理にフォーカスしたカードUI・タイムライン・モーダルなどの提案集です。進捗可視化やアクション導線の検討にご活用ください。

ギャラリーを開く

データダッシュボード

不適合情報ダッシュボード

不適合情報を整理した6つのダッシュボード案を掲載しています。サマリーカードやセクション分割など、レビュー用途に合わせて比較検討できます。

ギャラリーを開く

データダッシュボード

粉体情報ボード

粉体物性データを扱う5つのレイアウト案をまとめたギャラリーです。テーブル中心の構成からサイドバー切り替え型まで、分析シーンに応じた導線を確認できます。

ギャラリーを開く

データダッシュボード

粉体テストダッシュボード

試験スケジュールと粉体物性データを組み合わせた 3 つのレイアウト案です。期間フィルターやサマリーカードの密度を比較し、担当者レビューから会議共有までカバーします。

ギャラリーを開く
